通过 Stability Matrix 启动 WebUI 后，可以在网站里直接读取配置并出图。
  • 兼容 AUTOMATIC1111、Forge 与 ReForge 的 WebUI API

边界
暂不计划
保持个人项目
这些功能会明显增加运营、安全和维护成本，不符合目前的使用方式。
  • 公开注册、用户账号和社交关系
  • 社区投稿、公开作品流和内容审核
  • Scene 商店、订阅或付费服务
  • 长期托管的云端图库和生成服务
🌸 每次决定是否增加功能时，先检查四件事：
  • 有用优先 — 自己或朋友是否真的会用
  • 本地优先 — 数据能否留在自己的设备上
  • 安全分享 — 能否限制访问并随时停止
  • 内容优先 — 是否改善场景和出图，而不是只增加页面
