v0.5
文档与理念奠基
✓ 完成
世界观 + 哲学 + 美术规范 + 导演模式交互原型 + 色彩情绪 + 剧本模式
  • 世界观 / 哲学 / 美术规范 / 标签规范 / Prompt 规范 / Scene 规范 / 质量评分 / 路线 — 8 份 spec docs

  • 统一设计系统 (深灰底 + 樱花粉点缀)
v1.0
Prompt Builder v4
🚧 进行中
整合工具 + 数据。真正的"AI Galgame CG 创作工作台"。
  • 整合 Prompt_Builder v3 + director-flow 升级为 v4 (引导 / 速填双模式 + Scene 库 + 手动标签)
  • Scene 库 JSON 数据 (40/500 场景, 分类检索 + 搜索 + 季节/时段)
  • 历史记录 (localStorage, cap 30) + 一键导出 Prompt (.txt)
  • 参数 PNG 导出 (Canvas 重绘, 含 char / cfg / steps / sampler)
  • 实时预览图占位 (未来接 Stable Diffusion API)
  • 收藏夹 — 未完成
v2.0
Scene 库扩展
规划
从"工具"进化为"平台"。社区可以贡献 Scene。
  • Scene 库扩展到 500+ (多角色、多情绪、多季节)
  • Scene 分类检索 (情绪/场景/季节/适用)
  • 用户 Scene 提交 + 审核
  • 官方 Scene 满足 ⭐⭐⭐⭐⭐ 才能入库
v3.0
智能导演
规划
AI 辅助创作。输入一句话, 自动生成完整导演决策。
  • 输入一句话 → AI 建议情绪/镜头/构图/光照
  • LLM 自动写 Scene 故事描述
  • 风格迁移 (参考图 → 色彩/光照 Prompt)
v4.0
图片管理
规划
不只是生成, 还要管理。
  • 出图自动保存到本地 gallery
  • Prompt + 参数 + 图片 一键对比
  • 相似图检索 (根据 Prompt)
v5.0
LoRA 管理
规划
角色 LoRA 全生命周期管理。
  • LoRA 库浏览 (预览图 + 推荐参数)
  • 训练数据管理
  • LoRA 效果对比 (同一 Prompt, 不同 LoRA)
v6.0
ComfyUI 集成
远期
不只是 Prompt 工具, 而是完整 workflow 节点。
  • 一键导出 ComfyUI workflow JSON
  • ComfyUI 参数自动映射
  • Hires.fix / ControlNet / IP-Adapter 自动生成
v7.0
云同步 + 社区
远期
从个人工具进化为社区平台。
  • 用户账号 + 云端收藏/历史同步
  • Scene 官方商店 + 订阅
  • 创作者主页 + 作品集
  • 多语言 (中/英/日)
🌸 无论哪个版本, 都不违反 The Three Rules:
  • Scene First — 先有 Scene, 后有 Prompt
  • Character First — 人物永远是画面中心
  • Emotion First — 用户记住的不是 Prompt, 而是画面带来的回忆
