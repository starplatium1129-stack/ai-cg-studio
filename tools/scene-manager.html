📖 快速操作指南

🖼 替换场景样张

当你在导演工作台生成了比默认样张更好的图时：
1. 把新图片保存到 E:\code\2\lora\AI\SceneShowcase\2026-07-22_v14\images\
2. 文件名必须是 场景ID.jpg（如 sc001.jpg）
3. 运行 python scripts/build-scene-showcase.py 更新缩略图
4. 刷新场景库页面即可看到新样张

➕ 添加新场景

方法一（推荐）：点击上方「+ 新增场景」按钮，填写表单后保存，然后点「导出 JSON」下载文件覆盖 data/scenes.json

方法二：直接编辑 data/scenes.json，复制任意现有场景，修改字段即可。必填字段：
id, title, category, story, char, character, lora, emotion, season, time, timeOfDay, tags, mature, rating, prompt, negative, storyJa

🏷 添加新 Tag

1. 在「Tag 管理」标签页点击「+ 新增 Tag」
2. 输入英文名（用于 prompt）、中文名（用于显示）、分类
3. 系统会自动分配 ID
4. 导出 JSON 覆盖 data/tags.json

🔄 导出与保存

所有修改都保存在浏览器内存中，不会自动写入文件。
修改完成后，点击「📥 导出 JSON」下载文件，然后手动覆盖项目目录中的对应文件：
• 场景 → data/scenes.json
• Tags → data/tags.json

⚠️ 注意事项

• 场景 ID 必须唯一，格式为 sc001、sc002…
• Tag 英文名必须唯一，用于 prompt 生成
• char 字段决定角色 LoRA：nene → ayachi_nene_v11，natsume → shiki_natsume_v11
• dual_pose_verified 标签表示该双人场景有 OpenPose 骨架图
• 删除场景前建议先备份 scenes.json
